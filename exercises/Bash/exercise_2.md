## Instructions

Your folder structure should look something similar below.

```
python/
  |__hello_world/
      |__ README.md
      |__ LICENSE
      |__ requirements.txt
      |__ hello_world.py
      |__ docs/
            |__ doc_1.md
            |__ doc_2.md
            |__ doc_3.md
      hello_world_tests.py
      |__ hello/
            |__ world/
            |__ whoami
```

Let's make some changes (P.S. use the CLI).

```
python/
  |__spark/
      |__ README.md
      |__ LICENSE
      |__ INSTALLATION.md
      |__ requirements.txt
      |__ src/
          |__ spark/
                |__ away.py
                |__ tree.py
          |__ main.py
```

Now add this text to `README.md`.

```
# spark: A package that `sparks` your standard output
```

And this to `LICENSE`.

```
Copyright (c) 2024, [your name here]
```
